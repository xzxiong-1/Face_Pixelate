# Face_Pixelate

### Requirements
- Torch >= 1.0.0
- Torchvision >= 0.2.1
- Numpy >=  1.14.2
- opencv-python >= 4.0
- Matplotlib

Download weight from: https://pan.baidu.com/s/1Z1yNDotm1AH02c29Tal_ww [k9dk]

cd Face_Pixelate

pip install -r requirements.txt
